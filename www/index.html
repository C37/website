Softwares (CAD, CAM, Fresadora CNC)

Os requisitos para a operação dos softwares são:

  • Um computador (desktop e ou notebook) PC Mac ou Linux com 01 porta UBS disponível
  • Compátivel com os sistemas operacionais: Mac OS X v10.6.8 ou posterior, PC: Windows 7, Windows Vista, Windows XP Home ou Professional com Service Pack 3 ou posterior ou Linux: Ubuntu 10.04.4 LTS ou posterior
  • O Navegador Google Chrome instalado
